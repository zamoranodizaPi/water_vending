# Manual de Usuario

## Operación Básica
- Seleccione un producto desde la pantalla o mediante botones físicos.
- Ingrese crédito con el monedero.
- Cuando el crédito sea suficiente, siga las instrucciones visuales para colocar el envase.
- Presione `OK` para iniciar el flujo indicado.

## Crédito y Cambio
- El crédito se acumula automáticamente al detectar pulsos del monedero.
- Si el producto cuesta menos que el crédito disponible, la máquina calcula y muestra el cambio.

## Botones Físicos
- Botones de producto: seleccionan garrafón completo, medio o galón.
- `OK`: confirma la acción actual.
- Paro de emergencia: detiene el proceso durante llenado.

## Auditoría
- Existe modo de auditoría local protegido por código.
- El sistema muestra ventas, enjuagues, correos y resúmenes operativos.
- También se puede solicitar auditoría por correo autorizado.

## Solicitud por Email
- Si el asunto coincide con el nombre del sistema, se procesa la solicitud.
- Si el cuerpo contiene `auditoria`, responde con el reporte.
- Si el cuerpo contiene `log`, responde con el log semanal adjunto.

## Alertas
- Si el sensor detecta falta de agua, la máquina entra en `fuera de servicio`.
- Se muestra un mensaje de contacto y se envía una alerta por correo si está configurado.

## Configuración
- Desde el flujo de configuración se pueden cambiar precios, volúmenes, tema, nombre del sistema y contacto.

## GSM
- El proyecto no tiene integración GSM activa en esta versión `v1.0.0`.
